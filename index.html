What is the web?
The web is a collection of computers that are connected to each other. Some of this computers are what we call servers . These servers store the HTML documents , which are the documents that glues text, images and videos displayed on a website.

How does users access to content on different websites?
Websites are comprised of HTML documents. Users access to websites in the internet through their browsers. The computer uses HTTP protocol to ask the appropiate servers in the internet to send the HTML document of the URL the user has indicated. The browser is then in charged of reading, interpreting and displaying the HTML document.

HTML
HTML stands for Hypertext Markup Language. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". Tags and Elements HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags.
